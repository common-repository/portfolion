Creates a portfolio post type with category and tag taxomonies.

== Description ==

The Portfolion plugin creates a portfolio post type with a featured image to show off your work. Portfolion adds portfolio categories and tags specific to the portfolio post type. It adds custom post meta for a link to the portfolio project with several helpful functions for retrieval of the project link in your templates, similar to other WordPress template tags. Portfolion also comes bundled with starter templates to integrate into your theme. 

== Installation ==

1. Upload `portfolion` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I add a portfolio image? =

Use the native WordPress featured image feature to add your portfolio image.

= How do I use this with my theme? =

The easiest way is to create templates for your theme. Starter templates are available in the `templates` directory to get you started.

= How do I add the project link to my theme? =

There are several functions added to retrieve the project link similar to the other functions used when looping through posts. These functions are noted in the project link metabox.
